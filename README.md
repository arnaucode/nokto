## Nokto
XFCE Terminal Theme

### Terminal
To install, in the directory:
```
~/.local/share/xfce4/terminal/colorschemes/
```
Add the file:
```
nokto.theme
```

Then in the XFCE Terminal, go to Edit/Preferences/Colors/Presets, and select Nokto.


### Terminator
To install, in the directory:
```
.config/terminator/
```
Add the file 'terminator-config' under the name 'config'.
